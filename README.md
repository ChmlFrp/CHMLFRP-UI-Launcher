# **CUL-CHMLFRP启动器**
### 基于PyQt6开发的chmlfrp的第三方启动器图形化客户端

CUL原名CHMLFRP_UI

CUL顾名思义为CHMLFRP-UI-Launcher

下载地址: 蓝奏云：[https://cul.lanzoul.com/b00pzv3oyj](https://cul.lanzoul.com/b00pzv3oyj) 密码:ff50

下载解压运行即可(仅支持win10以上版本)，如有bug请提出谢谢!

肯定没bug，有的话找吃了
